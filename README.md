This Python program uses tkinter to create User Interface features, such as labels, textboxes, buttons, grid location, and more.
math is also used to perform specific calculations, such as math.sin, math.cos, and math.pi.
For this program, customization was also utilized for these features, like text, font, and height.
This program is a Projectile Estimator, in which the user types in a Velocity and Angle (which is turned into radians) 
and when the button, Calculate Distance, is clicked, a method, distanceTraveled, activates.
A label, lblDistance, has its text changed after the calculation that states the Distance Traveled.
